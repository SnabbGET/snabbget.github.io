Why use SnabbGET

SnabbGET is a graphic shell. It's more than a shell. It's a shell with a graphic interface. The graphic interface can be used by everyone. It's not only for programmers. If you never have opened a shell before, you can use SnabbGET. It's really simple to use. You can also use the shell in a basic terminal.
